#
 Amazon Analytics Project (2015–2025)

## Overview
End-to-end data analytics project analyzing Amazon-style e-commerce data
using SQL, Python, and Tableau.

## Tech Stack
- MySQL – Data storage & analytics
- Python – Data cleaning & EDA
- Tableau – Business dashboards
- GitHub – Version control

## Dashboards Created
- Executive Dashboard
- Festival Sales Analytics
- Demographics & Behavior
- Delivery Performance
- Seasonal Planning

## Key Insights
- Business revenue is shrinking YoY
- Decline driven by customer drop, not AOV
- Heavy dependence on limited categories
- Strong festival-driven revenue spikes

## Folder Structure
- `sql/` – Database & analytics queries
- `data/` – Cleaned CSV files
- `notebooks/` – Python EDA
- `tableau/` – Tableau dashboards

## Status
Core analysis completed. Advanced analytics in progress.
